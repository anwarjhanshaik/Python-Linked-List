# Python Singly Linked List (SLL)

Welcome to my repository! I am a beginner learning Data Structures and Algorithms in Python. This project is a completely custom implementation of a Singly Linked List built from scratch.

## 🚀 Features Implemented
* **Core CRUD Operations:** Insertions and deletions at the start, end, and specific positions.
* **Algorithmic Methods:** Fast & slow pointer techniques to find the middle node, detect loops, and automatically fix cycles.
* **In-Place Reversal:** Reversing a linked list with an optimal $O(1)$ space complexity.
* **Pythonic Magic Methods:** Added custom behavior for `__str__`, `__len__`, `__iter__`, and `__eq__` so the list works natively like Python's built-in lists.

## 🛠️ How to Run
Simply download or copy the `linked_list.py` file and run it using Python:
```bash
python linked_list.py
```
